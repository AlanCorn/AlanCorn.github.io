## 语法

elector {property: value}

## 选择器

1. 元素选择器
    如

    ```css
    div {property: value} 
    ```

2. 类选择器

    ```css
    .box {property: value}
    ```

    ```html
    <div class="box">
        ...
    </div>
    ```

3. ID选择器

    ```css
    #para1 {property: value} 
    ```

    ```html
    <p id="para1">
        ...
    </p>
    ```
